WHAT DATA IS STORED:

Each Drawing:
  - Artist name (your input)
  - Image data (canvas as PNG base64)
  - Vote count (starts at 0)
  - Created timestamp
  - Updated timestamp

Storage Options:
  1. In-Memory (default) - Works immediately, lost on restart
  2. MongoDB (optional) - Persistent, set up .env file
